# 🏆 Tournament Management System

A robust Java-based backend application designed to automate sports tournament administration, match scheduling, and real-time leaderboard management using MySQL.

## 🚀 Key Features
- **Dynamic Match Scheduling:** Automatically generates balanced fixtures and match pairings for any number of teams.
- **Real-time Leaderboard:** Implements ranking logic based on Wins, Draws, and Points, with secondary tie-breakers.
- **Database Persistence:** Fully integrated with MySQL using JDBC for secure and reliable data storage.
- **Automated Workflow:** Separate modules for creating tournaments, updating match results, and viewing standings.

## 🛠️ Technical Stack
- **Language:** Java (JDK 8+)
- **Database:** MySQL
- **Connectivity:** JDBC (Java Database Connectivity)
- **Architecture:** Modular Object-Oriented Programming (OOP)

## 📁 File Descriptions
- `tournament_creation.java`: Handles database setup, team registration, and initial schedule generation.
- `update_tournament.java`: Manages the input of match results and updates team statistics in the database.
- `view_tournament.java`: Fetches and displays the current tournament standings sorted by performance.
